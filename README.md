# DietHelper
>> 會員系統實作<br>
>> 資料庫查詢、儲存系統實作<br>
> - 可查詢食物營養素、登入會員紀錄身高體重與 BMI 變化 <br>
> - 實作語言: <br>
>| 普通版--`Python`+`SQLite`<br>
>| 網頁版--`Python`+`SQLite`+`HTML/CSS`+`Django`
> - 網路與資料庫程式設計課程之期中 & 期末專題
